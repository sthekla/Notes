# Entertainment Events Rebuild Plan

This repository now contains the strategic and technical foundation for rebuilding **entertainmentevents.com** into a modern, conversion-focused platform with full SEO optimization.

## Primary business goals

1. Sell tickets quickly and clearly.
2. Capture high-quality leads for show bookings.
3. Capture fundraiser booking inquiries for LNC.
4. Give the business team full control through an admin panel/CMS.
5. Build durable organic SEO growth over time.

## Recommended stack

- **Frontend:** Next.js (App Router) + TypeScript
- **Styling/UI:** Tailwind CSS + component library
- **CMS/Admin:** Sanity (or Strapi)
- **Database:** PostgreSQL (if custom booking/ticket inventory is needed)
- **Hosting:** Vercel + Cloudflare
- **Analytics:** GA4 + Google Search Console + Meta Pixel
- **Transactional:** Stripe (or preferred ticket provider integrations)

## Build phases

### Phase 1 — Foundation (1–2 weeks)

- Finalize architecture and content model
- Build conversion-first page templates
- Implement baseline technical SEO (meta, sitemap, robots, schema)
- Wire analytics + event tracking

### Phase 2 — Full content + feature rollout (2–4 weeks)

- Launch all core pages and forms
- Implement events and ticket workflows
- Implement booking pipelines (show + fundraiser)
- Publish initial SEO landing pages

### Phase 3 — Growth (ongoing)

- Publish weekly/monthly SEO content
- Build backlink/local authority
- A/B test CTA and form conversion rates
- Expand city and intent-specific landing pages

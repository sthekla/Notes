# Conversion and Funnel Plan

## Primary funnels

### Funnel A: Buy Tickets

1. Landing (Home/Event page)
2. Event detail
3. Ticket purchase click-through
4. Confirmation and follow-up retargeting

### Funnel B: Book a Show

1. Intent landing page
2. Trust block (videos/testimonials)
3. Short form (qualification fields)
4. Thank-you with scheduling CTA

### Funnel C: Fundraiser Booking (LNC)

1. Fundraiser explainer page
2. Case study/social proof
3. Fundraiser-specific inquiry form
4. Follow-up automation (email + call workflow)

## Conversion components

- Sticky CTA on mobile
- Multi-step forms for high-intent inquiries
- Social proof strips above fold
- Objection-handling FAQ section
- Strong above-the-fold value proposition

## Form strategy

- Keep required fields minimal initially
- Ask qualification details on step 2
- Use conditional logic by inquiry type
- Auto-route leads by form type and geography

## Tracking plan

- Track step-level form drop-off
- Track scroll depth on core landing pages
- Track click distribution on primary CTA buttons
- Track source/medium conversion performance

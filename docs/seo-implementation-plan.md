# SEO Implementation Plan

## 1) Technical SEO foundation

- Canonical tags on all indexable pages
- XML sitemaps split by type:
  - pages
  - events
  - blog
  - locations
- Clean robots.txt and crawl directives
- Open Graph and Twitter card metadata
- Dynamic metadata generation per page type
- Breadcrumb schema on deep pages
- Organization + LocalBusiness schema sitewide
- Event schema on event pages
- FAQ schema where relevant

## 2) On-page SEO standards

- One unique keyword target per page
- Unique title and meta description for every page
- H1/H2 hierarchy with semantic sectioning
- Internal links to related service and city pages
- Image alt text aligned to intent and accessibility
- FAQ block near bottom of key conversion pages

## 3) URL and keyword architecture

- Intent pages:
  - buy tickets
  - book entertainment show
  - book fundraiser entertainment
- Geo pages:
  - service + city
  - event type + city
- Content cluster pages:
  - educational booking guides
  - event planning checklists
  - fundraiser success stories

## 4) Performance and CWV targets

- LCP < 2.5s
- INP < 200ms
- CLS < 0.1
- Server-rendered primary content
- Optimized next/image usage and responsive images
- Font loading strategy (preload + swap)
- Third-party script budget and delayed loading

## 5) Measurement and reporting

- GA4 events:
  - ticket_click
  - booking_form_start
  - booking_form_submit
  - fundraiser_form_submit
  - phone_click
- Search Console monitoring:
  - index coverage
  - query/click trend by page type
  - page experience signals
- Monthly KPI dashboard:
  - organic traffic
  - conversion rates by funnel
  - top landing pages

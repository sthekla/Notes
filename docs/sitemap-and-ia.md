## Global UX rules

1. Persistent CTA bar on desktop and mobile:
   - Buy Tickets
   - Book a Show
   - Book a Fundraiser
2. Every page should include one primary CTA and one secondary CTA.
3. Contact options in header and footer (phone + inquiry form).
4. Schema-ready structured content sections on all long-form pages.
